Jarvis AI Voice Assistant

Overview

Jarvis AI Voice Assistant is a Python-based desktop voice assistant that performs automation tasks using voice commands. It can open applications, browse websites, take screenshots, detect faces, search files, and provide weather information.

---

Features

- Voice Command Recognition
- Open Google
- Open YouTube
- Open GitHub
- Open Calculator
- Open Notepad
- Open Command Prompt
- Open Paint
- Open Camera
- Take Screenshots
- Weather Information for Any City
- Face Detection using OpenCV
- File Search Functionality
- Text-to-Speech Responses

---

Technologies Used

- Python
- SpeechRecognition
- pyttsx3
- OpenCV
- Requests
- PyAutoGUI
- Pillow

Installation

1. Clone the Repository

git clone https://github.com/vaseema-shaik/jarvis-ai-assistant.git

2. Navigate to the Project Folder

cd jarvis-ai-assistant

3. Install Dependencies

pip install -r requirements.txt

---

Run the Project

python main.py
